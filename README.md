# Movie App 🎬

A React-based Movie Application that allows users to browse popular movies, search for movies, navigate through pages, and manage their favorite movies using TMDB API.

## Features

- Fetch and display popular movies from TMDB API
- Search movies by title
- Pagination for browsing multiple pages
- Add movies to Favorites
- Remove movies from Favorites
- Store favorites using Local Storage
- Dedicated Favorites Page
- Responsive User Interface

## Technologies Used

- React.js
- JavaScript (ES6+)
- TMDB API
- CSS
- Local Storage

## Installation

1. Clone the repository

```bash
git clone <your-repository-url>
```

2. Navigate to the project folder

```bash
cd movie-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
